{\itshape Macro} to take a semaphore from an I\+SR. The semaphore must have previously been created with a call to \hyperlink{semphr_8h_ae10bffadd26fbd5bcce76bf33a83ef30}{v\+Semaphore\+Create\+Binary()} or \hyperlink{semphr_8h_a7764616a918a46115403569a88148ad4}{x\+Semaphore\+Create\+Counting()}.

Mutex type semaphores (those created using a call to \hyperlink{semphr_8h_aa6a00aa9b91a9e5b3ebe4ae1c3f115c6}{x\+Semaphore\+Create\+Mutex()}) must not be used with this macro.

This macro can be used from an I\+SR, however taking a semaphore from an I\+SR is not a common operation. It is likely to only be useful when taking a counting semaphore when an interrupt is obtaining an object from a resource pool (when the semaphore count indicates the number of resources available).

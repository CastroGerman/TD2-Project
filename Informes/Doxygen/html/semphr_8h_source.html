290 /*
291  * xSemaphoreAltTake() is an alternative version of xSemaphoreTake().
292  *
293  * The source code that implements the alternative (Alt) API is much
294  * simpler because it executes everything from within a critical section.
295  * This is the approach taken by many other RTOSes, but FreeRTOS.org has the
296  * preferred fully featured API too. The fully featured API has more
297  * complex code that takes longer to execute, but makes much less use of
298  * critical sections. Therefore the alternative API sacrifices interrupt
299  * responsiveness to gain execution speed, whereas the fully featured API
300  * sacrifices execution speed to ensure better interrupt responsiveness.
301  */

76  * Para que el sistema se comporte como apropiativo, es decir que el squeduler dejara correr a la tarea con mayor prioridad que lo necesite
77  * ‘Fixed Priority Pre-emptive Scheduling with Time Slicing’
78  */
79 #define configUSE_PREEMPTION 1
80 /*
81  * Time slicing is used to share processing time between tasks of equal priority,
82  * even when the tasks do not explicitly yield or enter the Blocked state.
83  * Scheduling algorithms described as using ‘Time Slicing’ will select a new task
84  * to enter the Running state at the end of each time slice if there are other
85  * Ready state tasks that have the same priority as the Running task. A time
86  * slice is equal to the time between two RTOS tick interrupts.
87  *
88  * configUSE_TIME_SLICING en 1 indica que las tareas que compartan prioridad
89  * tambien compartiran el mismo tiempo de procesador (Se les deja correr el
90  * mismo tiempo a cada tarea).
91  */
92 #define configUSE_TIME_SLICING 1
93 /*
